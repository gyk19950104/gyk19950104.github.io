搭建博客

Fri, 20 Nov 2020

步骤

  • 1.申请GitHub账号并配置ssh
  • 2.brew安装hugo后选择主题
  • 3.熟悉常见hugo指令
  • 4.GitHub pages部署博客

指令

brew install hugo

hugo new site blog

git clone https://github.com/rmsubekti/the-roots-home.git themes/the-roots-home

hugo server -t the-roots-home --buildDrafts

hugo --theme=the-roots-home --baseUrl="https://gyk19950104.github.io" --buildDrafts

cd public
git init
git add .
git commit -m ''
git remote add origin https://github.com/gyk19950104/gyk19950104.github.io.git
git push -u origin master
